# Robot Tracker
This repository house the frontend application resposible to tracking a robot and executing instruction

## Getting Started

### Prerequisite
* Have Node installed
* Have an understanding off npm

Inside of the root of your project run the following to get started

* `npm i` this should install the modules required to run this application
* `npm run start` this will start up a local instance of this application in your browser on [local host](http://localhost:3000/)

## Live
If you want to test a live instance of this project follow [this link](https://robottracker.netlify.app/)
